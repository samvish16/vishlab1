        @font-face {
            font-family: 'CustomFont';
            src: url('custom-font.woff2') format('woff2'),
                 url('custom-font.woff') format('woff');
            font-weight: normal;
            font-style: normal;
        }

        body {
            font-family: 'CustomFont', sans-serif;
        }
    </style>
</head>
<body>
    <h1>Custom Font Delivery</h1>
    <p>This page demonstrates how to deliver a custom font to a client (browser) making a request to your server.</p>
    <p>To serve a custom font, you need to:</p>
    <ol>
        <li>Choose a font</li>
        <li>Create a custom font file in .woff or .woff2 format</li>
        <li>Upload the font file to your server</li>
        <li>Add a @font-face rule in your CSS to specify the font-family, and the path to the font file</li>
        <li>Apply the font-family to the desired elements in your HTML</li>
    </ol>
